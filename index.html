Решение задачи на определение простоты введенного числа

Определение необходимых переменных

Первоначальное значение переменной i равняется 2, тк это наименьший возможный натуральный делитель любого числа, отличный от 1. Переменной flag присваивается значение False, тк по умолчанию будем считать, что n - простое число

Решение циклом

В цикле пока переменная i не достигнет значение n // 2 (наибольшего возможного делителя любого числа), будут задействоваться условные операторы - if и else.
Так, если n делится без остатка на i, в консоли печается 'число не простое', и цикл завершает свою работу
В противном случае нужно прибавить к перменной i единицу. Далее работа цикла повторяется до тех пор, пока i не больше n // 2.
Далее проверяем, если значение переменной flag не изменилось (что означает, что делители числа не были найдены и число простое), печатаем "число простое".
